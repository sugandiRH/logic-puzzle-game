# 🧩 PuzzleX - Web-Based Logic Puzzle Game
PuzzleX presents players with mathematical and logical puzzles that must be solved to progress through multiple levels. The game includes a scoring system, life management, and player progression tracking to create an enjoyable learning experience.

## ✨ Features

- 🎮 Multiple puzzle levels
- ➕ Mathematical and logical challenges
- ❤️ Life management system
- 🏆 Score tracking
- 📈 Player level progression
- 👤 User registration and login
- ⏱️ Timer-based gameplay
- 📊 Scoreboard / Leaderboard
- 🔄 Dynamic level updates
- 🎨 Interactive user interface

## 🛠️ Technologies Used
  ### Frontend - HTML5, CSS3, JavaScript
  ### Backend - PHP
  ### Database - MySQL
  ### Development Environment - XAMPP, VS Code

## 🎯 How to Play

1. Register or log in.
2. Select a game level.
3. Analyze the puzzle.
4. Enter the correct answer.
5. Earn points for correct solutions.
6. Progress through higher difficulty levels.
7. Maintain lives to continue playing.
